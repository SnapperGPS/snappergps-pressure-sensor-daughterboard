## Flashing

* Go to [https://snappergps.info/flash](https://snappergps.info/flash).
* Connect the SnapperGPS receiver to your computer via USB.
* Pair the SnapperGPS receiver.
* Select the `SnapperGPS-PressureSensor` firmware.
* Flash the firmware.
